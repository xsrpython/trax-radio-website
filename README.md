## 🚀 Deployment

### Automated Deployment
```bash
cd trax_radio/website
./auto-deploy.bat
```

### Manual Deployment
```bash
cd trax_radio/website
git init
git add .
git commit -m "Update website"
git remote add origin https://github.com/xsrpython/trax-radio-website.git
git branch -M main
git push -u origin main --force
```
